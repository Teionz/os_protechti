# Ideias de Design - Sistema de Gestão de Ordem de Serviço ProTech TI

## Abordagem Escolhida: Dark Tech Professional

**Design Movement:** Cyberpunk Corporativo com influências de Design Systems modernos (Stripe, Linear, Vercel)

**Core Principles:**
1. **Eficiência Visual:** Cada elemento tem propósito claro; sem decoração desnecessária
2. **Contraste Inteligente:** Azul ciano (#00D9FF) como accent sobre fundo escuro (#0F1419)
3. **Hierarquia Clara:** Tipografia estratégica guia o usuário através dos formulários
4. **Responsividade Fluida:** Adapta-se perfeitamente de mobile a desktop

**Color Philosophy:**
- **Fundo Principal:** #0F1419 (Azul muito escuro, quase preto) - transmite profissionalismo e foco
- **Accent Primário:** #00D9FF (Ciano brilhante) - energia e confiança tecnológica
- **Accent Secundário:** #1E90FF (Azul royal) - hierarquia visual
- **Texto Principal:** #FFFFFF (Branco puro) - legibilidade máxima
- **Texto Secundário:** #A0AEC0 (Cinza claro) - informações auxiliares
- **Sucesso:** #10B981 (Verde esmeralda) - confirmações
- **Alerta:** #F59E0B (Âmbar) - avisos
- **Erro:** #EF4444 (Vermelho) - erros

**Layout Paradigm:**
- **Sidebar Esquerda:** Navegação persistente com ícones + labels
- **Conteúdo Principal:** Grid assimétrico com cards flutuantes
- **Formulários:** Multi-step com progresso visual; seções colapsáveis
- **Tabelas:** Horizontal scroll em mobile; sticky headers

**Signature Elements:**
1. **Glow Effect:** Bordas com brilho sutil (box-shadow com ciano)
2. **Cards Flutuantes:** Elevação com sombra profunda
3. **Indicadores de Status:** Badges coloridas com ícones (Pendente, Concluído, Cancelado)

**Interaction Philosophy:**
- Transições suaves (300ms) em hover/focus
- Feedback imediato em cliques (ripple effect)
- Tooltips informativos em ícones
- Confirmações antes de ações destrutivas

**Animation:**
- **Entrada:** Fade-in + slide-up (200ms) para cards
- **Hover:** Scale 1.02 + glow intenso para botões
- **Loading:** Spinner com rotação suave + pulsação
- **Transição de Página:** Fade-out/fade-in (150ms)

**Typography System:**
- **Display:** Poppins Bold 32px - Títulos principais
- **Heading 1:** Poppins SemiBold 24px - Títulos de seção
- **Heading 2:** Poppins SemiBold 18px - Subtítulos
- **Body:** Inter Regular 14px - Conteúdo principal
- **Small:** Inter Regular 12px - Labels e hints
- **Monospace:** Fira Code 12px - Números, IDs, valores monetários

---

## Justificativa

Este design reflete a identidade da ProTech TI: uma empresa de tecnologia moderna, confiável e profissional. O Dark Mode reduz fadiga visual em sessões longas de trabalho (comum em sistemas de gestão), enquanto o ciano brilhante cria um ponto focal que guia a atenção do usuário. A ausência de elementos decorativos mantém o foco na funcionalidade, essencial para um sistema B2B de gestão de ordens de serviço.
